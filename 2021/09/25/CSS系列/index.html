1.说说你对盒子模型的理解?

首先盒子模型是通过，border边框，外边距margin，内边距padding，内容content组成

盒子模型分为标准盒子和IE盒子

标准盒子的话就是width是通过width+margin+padding+border组成height和width类型

IE盒子的话宽通过width+margin组成

3.说说em/px/rem/vh/vw区别?

首先px是代表绝对单位，表示精度的像素

em和rem都代表的是字体的大小

vh和vw一般用于移动端，表示窗口的高度和窗口的宽度

4.css中，有哪些方式可以隐藏页面元素？区别?

首先有这么几种：display:none,visbillt:hidden,opacity:0，position:absolute

dispaly:none元素在页面上彻底消失，会触发重排和重绘

visbllt:hidden不会触发重排，会触发重绘

opacity:0不会触发重排，不一定触发重绘

position:abslote把元素移出可视区域

5.谈谈你对BFC的理解

首先BFC的话就是他是页面上一块渲染区域，有自己独立的容器空间，它与外界进行隔离，他是一个隔离的容器，容器里面的子元素不会影响到外面的元素

首先使用fiex布局，使用margin：auto，使用定位，top：50%，left：50%，tansform：tanslate（-50%，-50%）

使用table布局，使用grid布局然后子元素使用ju’stify-content：center，align-items:center

7.如何实现两栏布局，右侧自适应？三栏布局中间自适应呢？

俩栏布局的话使用浮动，左边栏使用float：left，右边栏使用margin-left撑出内容，给父元素添加BFC，overflow：hidden

还可以是有flex布局，父元素dispaly：flex，子元素去felx：1

  • 两边使用 absolute，中间使用 margin
  • 两边使用 float 和负 margin
  • display: table 实现
  • flex实现

8.说说flexbox（弹性盒布局模型）,以及适用场景？

flexbox就是弹性布局，可以简单快速的创建各种布局，自适应布局页面各种布局

主要属性有：

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

grid网格布局就是主要就是有列宽，和行高，通过clomus和rows，以及grad-gap来设置中间间距等等

10.CSS3新增了哪些新特性？

主要新增了阴影，圆角，过度，动画，转换，渐变等等

11.怎么理解回流跟重绘？什么场景下会触发？

首先得说一下浏览器的渲染机制，首先浏览器会解析html为dom元素，解析css为cssdom树，然后回结合dom树和cssdom树形成渲染树，之后根据返回的渲染树展示到页面，回流的回获取页面的节点信息，重绘就是获取节点的像素，传给一个叫页面

回流主要就是触发在添加删除元素，内容，大小尺寸，窗口发生改变回触发回流

重绘的话改变颜色，阴影，文本，等等回触发重绘

回流一定回触发重绘，重绘不会触发回流

解决办法避免使用table布局，避免使用css表达式，避免js操作css等等

12.什么是响应式设计？响应式设计的基本原理是什么？如何做？

响应式设计就是网络页面设计布局，可以达到一个自适应的形式

一般有几种方式使用媒体查询，也就media进行实现，通过媒体查询来改变不同大小屏幕分辨率显示的问题

还有使用vh，vw来改变窗口的大小

以及使用rem计量单位，他和px直接会有一个转换方式，1px等于16rem

以及使用百分比的形式实现自适应

13.如果要做优化，CSS提高性能的方法有哪些？

css一般都知道使用了写样式的，它一般有几种方式可以提高性能

合理使用选择器，就是避免使用伪类，通配符选择器

资源压缩等等，使用打包工具webpack来进行css的压缩等等
